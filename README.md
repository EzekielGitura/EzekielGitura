# Ezekiel Waweru

**Full-stack Developer | Cybersecurity Analyst | Technical Writer**

I build practical web systems that are fast, secure, and maintainable. My work sits at the intersection of software development, cybersecurity, and clear technical communication, which means I care about more than shipping features. I care about shipping systems people can trust.

I work mainly with the MERN stack, Python, cloud platforms, APIs, databases, and security tooling. I also write about software development, cybersecurity, and computer networking on Substack, where I break down technical concepts in a way that is useful for builders, learners, and security-minded teams.

---

## What I Bring

* **Secure full-stack development:** React, Node.js, Express, MongoDB, PostgreSQL, REST APIs, authentication, access control, and clean backend architecture.
* **Cybersecurity thinking:** SOC workflows, SIEM monitoring, log analysis, vulnerability assessment, IAM basics, incident response documentation, and GRC fundamentals.
* **Cloud and infrastructure awareness:** AWS, Azure, Linux, Docker, GitHub Actions, basic networking, and cloud security concepts.
* **Builder mindset:** I like turning unclear problems into working systems, documented workflows, and usable products.
* **Technical writing:** I explain software, security, and networking topics with clarity, structure, and enough depth to be useful.

---

## Current Focus

* Building secure full-stack applications with the MERN stack and Python.
* Strengthening DevSecOps, cloud security, and Kubernetes security fundamentals.
* Practising SOC analysis through labs, CTFs, SIEM projects, and incident response workflows.
* Exploring AI workflows, prompt engineering, LLM toolchains, and multi-agent collaboration.
* Writing technical articles on software development, cybersecurity, and computer networking.
